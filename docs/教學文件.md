# 教學文件

> Strategy snapshot: 2026-06-14
> 本文件以 SL-first 工作流為準；SAC 僅作研究 smoke/ablation。

## 1. 新研究流程

1. 準備資料。
2. 訓練或評估 SL h10 signal；h5 僅保留歷史比較。
3. 用 rule allocator 產生配置。
4. 跑 promotion gate。
5. 通過後才進 trade_guard dry-run。
6. dry-run 通過後才可討論 live。

## 2. SL 主線範例

```powershell
.\env\Scripts\python.exe -m sl_pipeline.walk_forward_sl --allocator rule --gate --seed 42
.\env\Scripts\python.exe experiment_report.py
```

看報告時優先檢查：

- OOS Sortino。
- Max drawdown，尤其 worst period。
- Turnover。
- Avg cash 與 cash 行為是否合理。
- 2024H2、2025H1 是否拖累整體結果。

## 3. SAC 研究範例

```powershell
.\env\Scripts\python.exe walk_forward.py --tier smoke --algo sac --cash-mode enabled
```

SAC smoke 的目標不是找可上線模型，而是快速排除錯誤方向。若模型全現金、單股集中、弱期崩壞或 turnover 爆炸，就停止該實驗。

## 4. Live 注意事項

目前沒有 SAC 模型可直接上線。任何 live 操作都必須先有 promotion gate APPROVED，再通過 trade_guard dry-run。Archive 內舊計畫均為歷史脈絡，不代表當前部署許可。
