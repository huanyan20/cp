## Ablation 結論

`ppo_enabled_with_features` 三個 seed 已補齊，但 overnight features 不應直接成為預設輸入：

| 指標 | With Features | Without Features | Delta |
|---|---:|---:|---:|
| Sortino | 0.87598 | 2.20340 | -1.32742 |
| Total Return | 1.079762 | 1.592630 | -0.512868 |
| Max Drawdown | 0.254605 | 0.361223 | -0.106618 |
| Turnover | 0.004470 | 0.031467 | -0.026997 |

解讀：overnight features 改善 drawdown 與 turnover，但明顯犧牲 Sortino、總報酬與勝率。此特徵目前只能視為風險抑制訊號，不能視為提升 alpha 的預設特徵。

## 建議決策

短期不要上線 promote。理由：

- 最佳候選的 worst-case MDD 仍高於 35%。
- overnight features 的 ablation 對 Sortino 是負貢獻。
- 2026H1 對部分 seed/模型貢獻過大，仍有 period concentration 風險。

下一個研究迭代應優先驗證 R4 效果，再考慮其他改善方向。

### R4 已實作內容（2026-06-08）

`trading_env.py` 做了以下最小 diff：

| 參數 | 舊值 | 新值 | 說明 |
|---|---|---|---|
| `LAMBDA_DRAWDOWN` | 0.3 | **0.8** | 回撤懲罰放大 2.7× |

**量化效果估算（DD=34%，股票曝險 93%）**：

| 懲罰項 | 舊版 | R4 |
|---|---|---|
| drawdown_p | `0.3×0.29=0.087` | `0.8×0.31=0.248` |
| regime_penalty | `0.5×0.93×0.24=0.112` | `1.0×0.93×0.26=0.242` |
| cash_defensive（7% cash） | 0 | `0.2×0.07×0.26=0.004` |
| **合計** | **≈0.20** | **≈0.49** |

舊版中一個 +0.5% 日報酬的 hybrid_reward（≈0.5）可以輕鬆蓋過所有懲罰，模型沒有學到降曝險的強信號。R4 下 DD=34% 時，若日報酬僅 +0.5%，reward ≈ 0.5 - 0.49 ≈ +0.01（幾乎為零），強迫模型認知「滿倉承受回撤代價高昂」。
